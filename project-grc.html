GRC AI

Governance, Risk & Compliance with Local AI.

Next.js 15 TypeScript Ollama (Gemma2) Tailwind CSS

Project Overview

GRC AI is a modern, AI-powered application designed to help organizations analyze policies, identify compliance gaps, and improve their regulatory posture. Unlike traditional cloud-based solutions, this project focuses on Data Privacy by performing all AI analysis locally on the device.

The system leverages Ollama running the Gemma2 2B model to interpret complex documents. It supports major frameworks like GDPR, ISO27001, HIPAA, and NIST, providing real-time risk scoring and actionable recommendations without any data leaving the secure environment.

Key Features

  • 100% Local AI: Zero cloud dependencies using Ollama & Gemma2 2B for maximum privacy.
  • OCR Integration: Advanced PDF analysis using pdf-parse to extract text from policy documents.
  • Gap Analysis: Automated identification of compliance gaps and risks across multiple frameworks.
  • Smart Dashboard: Visualizes real-time compliance metrics and trends using Recharts.

Technical Pipeline

  • 1
    PDF Upload: OCR extraction via pdf-parse.
  • 2
    Processing: Text cleaning and preprocessing.
  • 3
    AI Analysis: Local inference via Ollama (Gemma2 model).
  • 4
    Result Parsing: Structured JSON data extraction for the UI.
