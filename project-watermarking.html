Digital Watermarking Tool

Image Steganography & Attack Evaluation GUI.

Python Tkinter GUI OpenCV NumPy

Project Overview

This project is a comprehensive GUI-based tool developed for the "Information Hiding Techniques" course at Hashemite University. The main objective is to embed secret watermarks into images using various algorithms and evaluate their robustness against digital attacks.

The tool allows users to switch between Spatial Domain (LSB) and Frequency Domain (DCT, DWT) techniques. It also simulates real-world attacks like compression and noise to test if the watermark can still be recovered.

Key Features

  • Multi-Technique Embedding: Supports LSB (Spatial), DCT, and DWT (Frequency) methods.
  • Attack Simulation: Applies JPEG Compression, Blurring, Salt & Pepper Noise, Rotation, and Translation.
  • Quality Metrics: Calculates PSNR, SSIM, BER, and NC to measure image quality and extraction accuracy.
  • Interactive GUI: User-friendly interface to upload images, apply attacks, and visualize Original vs. Stego images.

Technical Pipeline

  • 1
    Embedding: The secret message is hidden into the image pixels (LSB) or coefficients (DCT/DWT).
  • 2
    Attacking: The watermarked image is subjected to noise or geometric distortions.
  • 3
    Extraction: The system attempts to retrieve the secret message from the attacked image.
  • 4
    Evaluation: Metrics (PSNR/SSIM) analyze the degradation, and (BER/NC) check message integrity.
